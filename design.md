# Empire CRM — Design System

## Brand
- Name: Empire CRM
- Tagline: The complete platform for professional DJs

## Color Palette
- Background: `#0D0F14` (deep dark navy)
- Surface/Card: `#151820` (slightly lighter navy)
- Surface2: `#1C2030` (card hover / elevated)
- Border: `#252A3A` (subtle border)
- Purple accent: `#7C3AED` (primary action)
- Purple light: `#9D6FEF` (hover states)
- Gold accent: `#F59E0B` (highlights, revenue, premium)
- Green: `#10B981` (success, confirmed)
- Red: `#EF4444` (danger, overdue)
- Blue: `#3B82F6` (info, calendar)
- Text primary: `#F1F5F9`
- Text secondary: `#94A3B8`
- Text muted: `#475569`

## Typography
- Headings: Syne (Google Font) — bold, geometric
- Body: Plus Jakarta Sans — clean, modern
- Numbers/mono: Geist Mono — sharp data display

## Layout
- Left sidebar: 240px fixed, collapsible to 64px
- Main content: fluid, max-width none
- Top header: 56px with breadcrumb + user avatar
- Stat cards: 4-column grid at top of dashboard
- Content sections: subtle card containers with border

## Component Style
- Cards: bg `#151820`, border `#252A3A`, radius 12px
- Buttons: purple primary, gold secondary
- Tables: dark striped rows, hover highlight
- Badges: colored pill badges per status
- Charts: purple/gold/green palette via Recharts

## Motion
- Page transitions: fade + slight translateY
- Sidebar: smooth width transition
- Cards: subtle scale on hover (1.01)
